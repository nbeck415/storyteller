Storyteller













fictionwriting.tech is a website that generates the beginning of a story based on a few words from user input. It uses Open AI’s GPT-2 model to build on the user input, and generates a short scene. We trained the model on stories from reddit’s /r/writingprompts using the praw library to scrape them. This project both demonstrates the voice and creativity of AI, which we hope will educate and inspire people.
